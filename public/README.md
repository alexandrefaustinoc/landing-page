# Pasta Public

Coloque sua foto de perfil aqui com o nome: **profile-photo.jpg**

## Instruções:

1. Renomeie sua foto para `profile-photo.jpg`
2. Coloque o arquivo nesta pasta (`public/`)
3. Formatos suportados: JPG, PNG, WEBP
4. **Tamanho recomendado para melhor qualidade:**
   - Mínimo: 800x800px (quadrada)
   - Ideal: 1200x1200px ou maior
   - Formato: Quadrada funciona melhor (1:1)
5. A imagem será automaticamente cortada em formato circular
6. **Qualidade:** Use imagens de alta resolução - o código está otimizado para manter a qualidade mesmo após o redimensionamento

## Dicas para melhor qualidade:

- Use imagens de pelo menos 1200x1200px
- Formato JPG com qualidade 90-100% ou PNG
- Evite comprimir demais a imagem antes de adicionar aqui
- A imagem será renderizada com alta qualidade usando técnicas de otimização CSS

## Alternativa:

Se preferir usar outro nome de arquivo ou formato, edite o arquivo:
`src/components/Hero.jsx` na linha 46 e altere o caminho da imagem.
